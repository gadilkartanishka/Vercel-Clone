Vercel Landing Page Clone

A front-end clone of the **Vercel landing page**, built to practice modern UI design and layout techniques. The project recreates the visual structure and styling of the Vercel website using core web technologies.

🛠 Tech Stack

* HTML
* CSS
* JavaScript

🚀 Getting Started

1) Clone the repository

```bash
git clone https://github.com/gadilkartanishka/Vercel-Landing-Clone.git
cd Vercel-Landing-Clone
```

2) Run the project

Simply open the `index.html` file in your browser.

🎯 Learning Objectives

This project was built to practice:

* Replicating real-world UI designs
* CSS layout techniques (Flexbox / Grid)
* Clean frontend structure
